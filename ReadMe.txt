1.Open Terminal
Win+R, then type cmd for Windows; Command+Space, then type Terminal for Mac.

2.Install Pygame
Enter the order below:
python3 -m pip install --user pygame
If not success, try:
python -m pip install --user pygame

3.Find the correct file path
Use cd to change the directory and ls to check files in current directory.
Type the order below to launch the game:
python3 alien_invasion.py 

Click 'Play' to start the game!
Press Space to shoot bullets.
Press Esc to quit.
Press F if you feel too hard. Be cautious!

Please contact the author if you encounter any bug or have any suggestion!
